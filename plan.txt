─────────────────────────────────────────────────────────────────────────
Был: рабочий MVP — один провайдер (FavoriteAPI), линейные настройки,

Станет (v2.0):
  • МНОГО ПРОВАЙДЕРОВ: FavoriteAPI, OpenRouter, FreeModel (переключение в 2 тапа)
  • ПОЛНОСТЬЮ КНОПОЧНЫЙ UX: меню, везде «Назад», «🏠 Меню», подтверждения,

  • i18n: русский + английский

3. КАРТА ЭКРАНОВ / КНОПОК (везде «Назад» + «🏠 Меню»)
─────────────────────────────────────────────────────────────────────────
/start → ГЛАВНОЕ МЕНЮ: ⚙️Настройки 🤖Провайдер ✏️Промпт 📣Каналы
                       📊Статистика ❓Помощь (🛠Админка — для админов)
Онбординг: провайдер → (URL) → ключ(проверка) → модель → промпт → сис.промпт
           → готово → привязка канала. На каждом шаге «Назад» и «Отмена».
Провайдер: список 3 шт. (✓ активный, ⚠️ FreeModel), смена ключа/базы/модели,
           🧪 тест соединения.
Промпт: 👁 показать, ✏️ изменить, 📚 пресеты, 🎨 сис.промпт вкл/выкл.
Каналы: список с тумблерами вкл/выкл, ➕ привязать (пересланный пост),
        ⏹ отвязать (с подтверждением), выбор активного.
Настройки: 🌐 язык ru/en, 👁 предпросмотр, 🎨 сис.промпт, ♻️ сброс контекста,
           🗑 сброс статистики, ярлыки в провайдер/промпт/каналы.
Статистика: обработано/ошибки/ср.время/последняя активность/контекст_kb.
Админка: 👥 юзеры (список/просмотр/бан), 📊 глобальная стата, 📢 рассылка,
         📜 логи.
Предпросмотр: ИИ-результат в ЛС → ✅ Опубликовать / ❌ Отклонить / ✏️ Правка.

      response_count, last_post_at)

Миграция: старая одно-табличная users → users + provider_configs(favoriteapi)
+ channels. Бэкап bot.db→bot.db.bak. Идемпотентно.
